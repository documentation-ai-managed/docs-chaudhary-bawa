<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]} id="v1-2-0">

## New Features

- Added real-time collaboration: Multiple users can now edit documents simultaneously with live cursors and conflict resolution.
- Introduced version history: Roll back to previous document versions with one click.

## Improvements

- Enhanced search performance by `>50%` for large documentation spaces.
- Updated UI components for better mobile responsiveness.

## Bug Fixes

- Fixed markdown rendering issues with nested lists.
- Resolved authentication token refresh errors during long sessions.

</Update>

## Upgrade Instructions

Follow these steps to update Chaudhary Bawa to the latest version.

<Steps>
  <Step title="Check Current Version" icon="check-circle">
    Run the version check command in your project root.
    
    <CodeGroup tabs="npm,yarn">
    ````bash
    npm run version
    ````
    ````bash
    yarn version
    ````
    </CodeGroup>
  </Step>
  
  <Step title="Update Dependencies" icon="download">
    Install the latest release.
    
    <CodeGroup tabs="npm,yarn">
    ````bash
    npm install chauhary-bawa@latest
    ````
    ````bash
    yarn add chauhary-bawa@latest
    ````
    </CodeGroup>
  </Step>
  
  <Step title="Restart Server" icon="refresh-cw">
    Restart your development server to apply changes.
    
    ````bash
    npm run dev
    ````
  </Step>
</Steps>

<Callout kind="alert">
  Always back up your documentation space before upgrading, especially with breaking changes in `v1.0.0`.
</Callout>
