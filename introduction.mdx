## Overview

Chaudhary Bawa provides a centralized space to create, organize, and manage your project documentation. You can build wikis, API references, and guides with real-time collaboration, version control, and powerful search. Start documenting your projects today and keep your team aligned effortlessly.

## Quick Start

Get up and running with Chaudhary Bawa in minutes.

<Steps>
  <Step title="Create Your Space" icon="plus">
    Sign up at `https://dashboard.example.com` and create a new documentation space for your project.
  </Step>
  <Step title="Add Your First Page" icon="file-plus">
    Click "New Page" and start writing using the built-in editor.
  </Step>
  <Step title="Publish and Share" icon="share-2">
    Publish your page and share the link with your team. Customize access levels as needed.
  </Step>
</Steps>

<Callout kind="tip">
  Customize your space with the brand color `#3B82F6` in settings for a consistent look.
</Callout>

## Brand Guidelines

Follow these guidelines to maintain visual consistency across your documentation.

Use the primary brand color `#3B82F6` for headings, buttons, and accents.

```css
:root {
  --brand-primary: #3B82F6;
  --brand-primary-hover: #2563EB;
}
```

```css highlight="1-2"
button {
  background-color: var(--brand-primary);
  border-color: var(--brand-primary);
}

button:hover {
  background-color: var(--brand-primary-hover);
}
```

## Embed Examples

Embed your Chaudhary Bawa pages in other sites using this simple code.

<CodeGroup tabs="HTML,React">
  ````html
  <iframe
    src="https://docs.example.com/your-space/embed/page-id"
    width="100%"
    height="600"
    frameborder="0">
  </iframe>
  ````

  ````jsx
  import { Embed } from '@chaudhary-bawa/embed';

  function DocsEmbed() {
    return (
      <Embed
        spaceId="your-space-id"
        pageId="your-page-id"
        height={600}
      />
    );
  }
  ````
</CodeGroup>
